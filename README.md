# AACS TERMINAL
![AACS LOGO](./app/RKN.png) 
AACS terminal - небольшой проект ко дню рождения подруги, выполненный в стиле ARG, идея [ARRS TERMINAL](arrs.host)


### Визуальная часть
Подложка сайта это видео сделанное с нуля с помошью After effects
Логотип и символика сделана с помошью Photoshop
Идея для логотипа - Логотип Роскомнадзора
Идея сюжета - Тирания РКН'а в отношении рунета, массовые блокировки ресурсов, недавнее убийство сотрудника РКН


### Техническая часть
Весь сайт написан на JavaScript
При разработке использовались модули JS
Многие фичи были реализованы с поддержкой ИИ
